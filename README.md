# Digital Portfolio

A standalone, interactive portfolio — separate from the Fair Lending AI Validation project.

## How to view

1. **Open in browser**  
   Double-click `index.html` or open it from your file manager.  
   Or from a terminal in this folder:
   ```bash
   # Windows
   start index.html

   # macOS
   open index.html

   # Or use a simple local server (recommended for fonts/links)
   npx serve .
   ```
   Then visit the URL shown (e.g. http://localhost:3000).

2. **Edit content**  
   - **index.html** — Name, tagline, experience, projects, contact links, location  
   - **styles.css** — Colors, spacing, typography  
   - **script.js** — Nav scroll highlight, mobile menu

No build step or Node.js required. Works offline once loaded.
